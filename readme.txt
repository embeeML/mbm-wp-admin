A WordPress admin toolkit plugin.

== Description ==

MBM WP Admin Toolkit provides a foundation for building powerful WordPress admin functionality.

== Installation ==

1. Upload the `mbm-wp-admin-toolkit` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
